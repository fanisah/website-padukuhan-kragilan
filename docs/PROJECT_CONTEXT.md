# Project

Website Sistem Informasi Padukuhan Kragilan

# Purpose

A simple CMS-driven village information website developed for KKN UGM.

Target users:

- Village administrators
- PKK
- Community members

Main objectives:

- Publish village information
- Manage UMKM
- Publish news
- Publish agenda
- Publish gallery
- Easy administration

# Technology

## Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

## Backend

- Supabase
- PostgreSQL
- Authentication
- Storage (planned)

## Deployment

- Vercel

# Development Principles

- Keep UI simple.
- No unnecessary dependencies.
- Preserve existing design.
- Prefer reusable components.
- Separate UI from service layer.
- Run `npm run build` before completing implementation.
